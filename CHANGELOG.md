# Changelog

All notable changes to DesignForge will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure
- Core agent implementation with Claude AI integration
- MCP server support for Figma and Design System
- CLI with start, watch, interactive, debug commands
- Configuration file support (designforge.config.js)
- Environment variable configuration
- Comprehensive documentation
- Basic and advanced usage examples
- Test suite foundation
- Monorepo structure with workspaces

### Features
- Autonomous Figma to code generation
- Design system component mapping
- TypeScript/React code generation
- Automated test generation with Jest
- Storybook story generation
- Design-dev parity validation
- Gap analysis for design system components
- Real-time progress tracking
- Multi-turn conversation with Claude
- Verbose logging mode
- Dry-run mode for previewing

### Documentation
- Getting Started guide
- Configuration guide
- Workflows documentation
- API reference
- Contributing guidelines
- Example projects

## [0.1.0] - 2025-01-22

### Added
- Initial alpha release
- Core functionality for Figma to code conversion
- Basic CLI interface
- Documentation structure
- Example projects
